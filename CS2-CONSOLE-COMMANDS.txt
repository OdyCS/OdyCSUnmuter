ODYCSUNMUTER BETA — CS2 CONSOLE COMMANDS
===========================================

Run these commands before playing the rewritten demo (-1 should be replaced by the specific binary value of the player(s) desired if you don't want to hear everyone):

    tv_listen_voice_indices -1
    tv_listen_voice_indices_h -1

WHAT THEY DO
------------
tv_listen_voice_indices -1
tv_listen_voice_indices_h -1
    Enables all low and high SourceTV player-slot voice-listener bits.
    Together they cover up to 64 slots.

OPTIONAL DISPLAY COMMAND
------------------------
    cl_sanitize_muted_players 0

This only affects whether muted players' names and avatars are hidden.
It does NOT restore audio and normally is not required for an
OdyCSUnmuter output.

HOW TO LOAD THE DEMO
--------------------
Load the generated *_OdyCSunmuted.dem the same way you normally load a demo.

If playdemo cannot find it, copy the generated file into your CS2 game\csgo
folder, then run:

    playdemo demo_filename_OdyCSunmuted
