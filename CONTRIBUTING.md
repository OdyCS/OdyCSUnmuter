# Contributing

Bug reports and CS2 compatibility findings are welcome.

Include:

- OdyCSUnmuter version
- demo source
- exact output
- whether the original demo plays correctly
- whether the target has non-zero audio bytes
- whether the recommended CS2 console commands were used

Do not attach private demos without authorization.

Changes must never overwrite the original demo and must preserve consistent
identity remapping, Ody's license, and third-party attribution.
